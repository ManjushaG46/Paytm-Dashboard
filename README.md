# Paytm-Dashboard

# Project Title
Paytm Transaction Analytics Dashboard using Power BI
Project Purpose
The purpose of this project is to analyze digital payment transactions and revenue trends using an interactive Power BI dashboard. The dashboard helps in understanding customer behavior, device usage, transaction types, and city-wise performance to support data-driven business decisions.

# Technologies Used
Power BI Desktop
DAX (Data Analysis Expressions)
Power Query
Data Modeling
Interactive Data Visualization

# Data source - Kaggle

# Highlights - 
KPI-driven Power BI dashboard
City, device, and transaction-type analysis
Time-based revenue insights
User behavior and retention analysis
Interactive and performance-optimized visuals
